# WhiteboardTogether ![Static Badge](https://img.shields.io/badge/IN%20DEVELOPMENT-081461?style=for-the-badge)
A real-time collaborative web application allowing users to draw, and share with others using Node.JS and React.

## Overview and Features
- A digital whiteboard that used real-time communication to draw with others.
- Can be used for multiple scenarios such as meetings, brainstorming, and more!
- Multiple color and brush options.
- Saving and loading drawings. Your drawings will also save in a format that is something similar to a .png, but that is undecided.
  
## Tech Stack
- Client (Frontend): Node.js with React |
This frontend is used to build and produce a modernized interface and whiteboard canvas.
- Backend: Node.js |
This backend is used with socket.io to handle real-time requests and send drawings to users' interfaces almost instantly.
